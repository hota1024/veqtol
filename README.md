## ⚙ Features

- 🚀 Generate static blog HTML files.
- 📕 Rich markdown by MDX.
- 🏷️ Easy tags.
- 📚 Posts list with pagination.
- 🎨 Simple UI and UX.
- 🐯 Powerful code block with Prism.js, titled code and highlight line query.
- 🔧 Customizable for your site.
